# b.well Mobile Embeddable Examples

Example mobile apps demonstrating how to render b.well's embeddable solution via WebView with JavaScript injection for user authentication.

## Projects

| Platform | Directory | Status |
|----------|-----------|--------|
| Android | [`bwell-embeddable-android/`](./bwell-embeddable-android/) | Available |
| iOS | `bwell-embeddable-ios/` | Coming soon |

## Overview

These examples show how to:
- Load the b.well embeddable in a native WebView
- Inject JavaScript to authenticate users via `bwell.setUserToken()`
- Handle initialization state and errors

See individual project READMEs for setup instructions.
